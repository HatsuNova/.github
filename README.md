# SuzuYoru 铃夜开发组

## 组织简介

**铃夜开发组**是**中国计量大学（CJLU）Galgame 同好会**的官方开发部门，负责面向 Galgame 社群及相关场景的软件与内容开发。

## 开发方向

开发组的工作范围包括但不限于 Web 应用、软件与开发工具、程序逆向分析与兼容性研究、游戏开发，以及其他能够提升效率的工具（例如远程唤醒工具）。程序逆向分析与兼容性研究仅在获得授权或适用法律允许的范围内开展。

## 当前项目

目前多数项目由会长 [ArisuMika](https://github.com/ArisuMika520) 主导开发。

### CJLUGalOS

中国计量大学 Galgame 同好会的官方网站与社区平台：[cjlugal.moe](https://www.cjlugal.moe)。平台覆盖 Web 端与 App 端，以社区服务为核心，提供公告、活动、投票、聊天和游戏等功能。

#### 源码与授权说明

除另有明确标注的第三方组件或素材外，CJLUGalOS 的核心源代码目前不对外公开。未经权利人事先授权，请勿复制、修改、分发、二次发布或将其用于衍生项目；第三方组件与素材应遵循各自的许可条款。

### AutoWoL

基于 Wake-on-LAN（WoL）的远程唤醒工具：通过局域网内的常驻设备向目标主机发送 WoL 魔术封包，以实现远程开机。使用前需确认目标主机、网卡与固件支持 WoL，且已完成相应的系统与网络配置；常驻设备通常需要保持可用。

## 评估迁移的项目

### ECampusElectricity

面向易校园宿舍电费服务的查询与提醒工具，支持电费余额查询、余额预警（邮件通知）及 QQ 机器人查询。该项目目前针对中国计量大学进行了适配；部署时仍需根据实际环境完成账户与通知渠道等配置。

### YunMei Air

“云莓不智能”服务的渐进式 Web 应用（PWA）重实现，用于在已获授权的范围内快捷调用宿舍门锁相关功能。项目当前仍存在稳定性与兼容性问题，待完善后再决定是否迁移；同时计划评估替换 Next.js 的实现方案。

### GalFetch

Galgame 元数据聚合工具，可汇集 VNDB、Bangumi、Steam 与 DLsite 等来源的信息，用于快速建立轻量级 Galgame 数据库。当前条目匹配、冲突处理与去重逻辑仍待优化。
